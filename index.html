What transfer_stake means
Alpha moves between hotkeys — no AMM, no TAO conversion
Unlike removeStake (which burns alpha through the AMM to receive TAO), transfer_stake moves alpha directly between coldkey/hotkey pairs. The alpha stays in the ecosystem. TAO value shown (1,853 τ) = the TAO-equivalent of the alpha at the time of each transfer, not TAO actually received.
Why daily distributions
Automated validator dividend model — consistent with professional subnet ops
The 596 transactions split evenly across ~306 days (≈2 txns/day: one to each hotkey) match a scheduled daily payout structure. This is consistent with a subnet operator running an automated validator rewards system, not manual extraction behavior.
